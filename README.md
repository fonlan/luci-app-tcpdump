# luci-app-tcpdump

LuCI interface for tcpdump.\
It can be used to capture live ethernet traffic for analysis.

## How to build into firmware:

`git clone https://github.com/fonlan/luci-app-tcpdump.git ./package/luci-app-tcpdump`

`make menuconfig`
